# AcWingOS 任务栏小说阅读器

### 使用方法

| 快捷键  |            用途            |
| :-----: | :------------------------: |
| alt + \ |       从本地上传书籍       |
| alt + / |      开始阅读或老板键      |
| alt + , |           上一页           |
| alt + . |           下一页           |
| alt + ; | 从 localStorage 中选择书籍 |

上传时自动检测和转换编码 ( 可能要花费一点时间，可以本地转换成 **UTF-8** 后上传 )
